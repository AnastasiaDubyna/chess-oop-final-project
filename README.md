## Online chess game for OOP course final project
## Install
`pip3 install -e .`
## Install Dev
`pip3 install -e ".[dev]"`
## Run
`pserve development.ini --reload`
OR
`python3 -m pyramid.scripts.pserve development.ini --reload`
